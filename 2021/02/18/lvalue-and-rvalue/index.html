Example

lvalueе’Ңrvalueе…·дҪ“зҡ„е®ҡд№үжҳҜд»Җд№Ҳ?

е®һйҷ…дёҠlvalueе’ҢrvalueеҫҲйҡҫз»ҷеҮәдёҖдёӘе…·дҪ“зҡ„е®ҡд№үпјҢдҪҶжҳҜжҲ‘д»¬еҸҜд»Ҙз»ҷеҮәз®ҖеҚ•зҡ„,жҰӮжӢ¬зҡ„,еә”еҜ№еӨ§йғЁеҲҶжғ…еҶөзҡ„иҜҙжҳҺгҖӮ

  • lvalue : дёҖдёӘеҜ№иұЎеңЁеҶ…еӯҳдёӯеҚ жҚ®дәҶеҸҜиҜҶеҲ«дҪҚзҪ®зҡ„пјҢжңүең°еқҖзҡ„пјҢжҢҒд№…зҡ„(и„ұзҰ»дәҶеҚ•дёӘиЎЁиҫҫејҸиҜӯеҸҘиҝҳдҫқз„¶еӯҳеңЁзҡ„).
    • дҫӢеҰӮ: дёҠйқўжҸҗеҲ°зҡ„int a = i+1,зЁӢеәҸиҝҗиЎҢеҲ°дәҶдёӢдёҖжқЎиҜӯеҸҘaдҫқз„¶еӯҳеңЁеҶ…еӯҳдёӯдёәпјҢ&aжҳҜеҗҲжі•зҡ„пјҢlvalueпјҢi+1еҲҷдёҚеӯҳеңЁеҶ…еӯҳдёӯпјҢдёҚжҳҜжҢҒд№…зҡ„пјҢ&(i+1)жҳҜдёҚеҗҲжі•зҡ„пјҢжүҖд»ҘдёәrvalueгҖӮ
  • rvalue : иЎЁиҫҫејҸдёӯд»»дҪ•дёҚжҳҜlvalueзҡ„еҜ№иұЎе°ұжҳҜrvalue.

дёӢйқўи§ЈйҮҠеҮәиҮӘLei Maoзҡ„еҚҡе®ўlvalue VS rvalue, rvalue Reference, and Move Semantics in C++:

lvalue expression is associated with a specific piece of memory, the lifetime of the associated memory is the lifetime of lvalue expression, and we could get the memory address of it. rvalue expression might or might not take memory. Even if an rvalue expression takes memory, the memory taken would be temporary and the program would not usually allow us to get the memory address of it.
